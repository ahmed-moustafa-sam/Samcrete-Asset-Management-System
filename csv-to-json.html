أداة تحويل CSV إلى JSON

كيفية الاستخدام:

  1. اضغط "تصدير CSV" في لوحة التحكم الرئيسية
  2. افتح الملف المُنزل في Excel أو أي محرر نصوص
  3. انسخ محتوى الملف والصقه في حقل "CSV" أدناه
  4. اضغط "تحويل إلى JSON"
  5. انسخ الناتج من حقل "JSON"
  6. استبدل محتوى ملف assets.json في GitHub بالناتج

💡 نصيحة مهمة:

يمكنك الآن تفعيل الحفظ التلقائي على GitHub من إعدادات GitHub في الواجهة الرئيسية!

بهذا لن تحتاج لإعادة التحويل والرفع يدوياً عند كل تعديل.
