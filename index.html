إعداد الاتصال بـ OneDrive

للربط مع OneDrive تحتاج إلى تسجيل تطبيق في Azure:

  1. اذهب إلى Azure App Registrations
  2. اضغط "New registration" → أدخل اسم التطبيق
  3. اختر "Accounts in any organizational directory and personal Microsoft accounts"
  4. Redirect URI: اختر "Single-page application (SPA)" وأدخل عنوان موقعك
  5. انسخ الـ Application (client) ID
يجب أن يتطابق مع الـ Redirect URI المسجل في Azure
